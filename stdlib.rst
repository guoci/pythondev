+++++++++++++++++++++++
Python standard library
+++++++++++++++++++++++

Removed modules
===============

* Python 3.10

  * ``formatter``
  * ``parser``

* Python 3.9

  * ``_dummy_thread``
  * ``dummy_threading``

* Python 3.8

  * ``macpath``

* Python 3.7

  * ``fpectl``

* Python 3.6

  * ``CDIO``
  * ``CDROM``
  * ``DLFCN``
  * ``IN``
  * ``STROPTS``
  * ``TYPES``

* Python 3.0

  * ``bsddb3``
  * Many old modules were removed: see https://www.python.org/dev/peps/pep-3108/#modules-to-remove Examples:
  * ``md5``
  * ``mimetools``
  * ``posixfile``
  * ``rfc822``
